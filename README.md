# Portfolio Pulse

2026 투자학교 학생들이 각자 포트폴리오를 입력하고, 미국주식/한국주식/암호화폐 시세와 재무정보를 확인할 수 있는 웹앱입니다.

## 추천 배포 방식

이 앱은 웹앱 + PWA 방식입니다. 학생들은 링크로 접속하고, 스마트폰에서는 홈 화면에 추가해서 앱처럼 사용할 수 있습니다.

학생의 포트폴리오 데이터는 현재 각자의 브라우저 `localStorage`에 저장됩니다. 그래서 별도 회원가입이나 DB 없이 바로 배포할 수 있습니다.

## 로컬 실행

```powershell
npm start
```

브라우저에서 `http://localhost:4173`을 엽니다.

## 배포

Node.js 서버를 실행할 수 있는 서비스에 올리면 됩니다.

추천 순서:

1. Render
2. Railway
3. Fly.io
4. VPS

Render 기준:

- Build Command: 비워두거나 `npm install`
- Start Command: `npm start`
- Node version: 20 이상

배포 후 학생들에게 생성된 HTTPS 링크를 공유하면 됩니다.

## PWA 설치

스마트폰에서 배포 링크 접속 후:

- iPhone Safari: 공유 버튼 -> 홈 화면에 추가
- Android Chrome: 메뉴 -> 앱 설치 또는 홈 화면에 추가

## 데이터

- 시세/차트: Yahoo Finance 공개 차트 엔드포인트
- 미국 종목 재무정보: Yahoo Finance quoteSummary
- 한국 종목 재무정보/시가총액: 네이버 금융
- 암호화폐: Yahoo Finance 심볼 형식 예: `BTC-USD`, `XRP-USD`

공개 데이터는 제공처 정책, 지역 네트워크, 장 운영 시간에 따라 지연되거나 일시적으로 제한될 수 있습니다.
